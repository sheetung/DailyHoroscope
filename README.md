# DailyHoroscope

Daily Horoscope for LangBot

### Usage Instructions

After installing via the LangBot Plugin Market, you can get today's fortune by sending the command 运势 (Yùnshì).
Currently, you can only get your fortune once per day

#### !!API Token

This is a small server. To prevent abuse, please contact the plugin author to obtain the API token.
