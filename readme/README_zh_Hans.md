# DailyHoroscope

Daily Horoscope for LangBot

赛博每日运势，纯随机，图一乐

### 使用说明

通过LangBot插件市场安装后，可以通过命令`运势`获取今日运势。

目前每日仅限获取一次运势

#### !!api token

小服务器，为了防止滥用，apitoken请联系插件作者获取
